RASCAL (Robotic Autonomous Scale Car for Adaptive Learning) is a custom 3d-printed scale car platform built for autonomous vehicle research.

Components:

  • Software framework created with Python and ROS to enable high-level control of RASCAL. Covers low-level sensor and actuator control through Arduino and USB Serial.

  • Custom design to implement ackerman steering and rear differential on a 14:1 scale with 3d printing.

  • Full data pipeline for CNN model for autonomous driving
    • Data processing from raw position and control signals
    • Custom replay viewer to verify, edit, and remove data
    • Data augmentation using image skewing to approximate homography
    • Hyperparameter optimization using Dragonfly
    • Model takes camera data and outputs lookahead points which are aggregated to form a spline, followed by pure pursuit control.

  • ROS Web Display - a modular GUI created with Flask and JavaScript
    • Enables real-time control with joystick integration
    • 2d data visualization and capability for interactive applications such as path planning
    • Live camera feed
    • 3d perspective visualization using perspective warping, overlaid on the camera
    • remote execution of user-defined commands
    • Button mapping from joystick to commands
    • fuzzy search of commands, command pinning

  • Utilized an efficient workflow on remote Linux machines using SSH, terminal-based editors, and shell commands

  • Presented technical progress, methodology, and significance in weekly updates and a final open-house event

The Recyclinator is a machine that automatically sorts recyclable objects when placed in its carriage. The device uses multiple sensors to determine the material of the object, traverses on a track to the corresponding bin, and opens to deposit the object in the correct bin.

Uses an arduino to control stepper motors, capacitance sensor, and servo to sense, move along a track, and open a door

Integration with a flutter app for a phone attached to the machine to provide a User interface, as well as use the camera for object detection and light obstruction detection. The app communicates with the arduino using USB serial communication.

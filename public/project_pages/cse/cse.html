The Cougar Script Editor is a robust system developed for FRC Team 1403 to provide a user interface for encoding sequences of autonomous actions. The editor is a drag and drop graphical user interface that enables of creation and modification of cougar scripts, while showing a preview of the projected actions of the robot. This script can be saved and exported, interpreted, and executed by the robot. The system is designed to be easily adaptable to fit any time of game, field, robot, or command!

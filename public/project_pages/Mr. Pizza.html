A full-stack REST-based application with a client, custom Node.js server, and a MySQL database to enable pizza ordering, delivery tracking, and management tools for backend operations and order processing.

Components:

  • Custom server framework written from scratch in Node.js
    • Modular routing framework for serving static files and API URIs
    • Database setup script generation for production and test environments
    • Password encryption with Bcrypt
    • Encrypted token-based user and role authentication

  • Google Maps Integration:
    • Validation of delivery and store addresses
    • Calculation of optimal pizza delivery route and assignment
    • Automated directions for delivery drivers
    • Delivery time estimation for customers

  • Stripe API integration through webhooks for streamlined payment.

  • Fine-tuned NLP.js chatbot to respond to inquiries about business policies and restaurant information

  • Webscraper written using Puppeteer.js to collect example data for menu
